# Cybersecurity & IT Home Lab

A physical home lab built from scratch for career development in cybersecurity, networking, IT administration, and helpdesk support. Every phase is documented with screenshots, commands, configurations, and troubleshooting steps.

## Hardware
- **Server:** Alienware Aurora R12 (Intel i5-11400F, 32GB DDR4) running Proxmox VE 9.1
- **Workstation:** HP ZBook 16 G11 (management and attack machine)
- **Network:** TP-Link TL-SG108E managed switch

## Project Phases

|---|---|---|
| 1 | Proxmox Hypervisor + Active Directory | Complete |
